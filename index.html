User Manual

Welcome to Divine Counter.

1. Daily Reset System

The "Daily Session" and "Today's Rounds" reset to zero automatically at midnight. Lifetime stats are preserved.

2. The Dashboard Metrics

  • Daily Session (Center): Taps performed today.
  • Today's Rounds (Left): Daily Session / 108.
  • Lifetime Taps (Right): Cumulative total.
  • Lifetime Rounds (Right Sub): Lifetime Total / 108.

3. Managing Mantras

  • Add: Open Menu, enter Name/Goal, Click Add.
  • Delete: Click Trash icon twice.

4. Reset Options

  • Reset Round: Zeros Daily Session.
  • Reset Total: Clears Lifetime Stats.
  • Wipe All: Factory reset.
